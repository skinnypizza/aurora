# Contenido para OpenCollective

## Título corto (tagline)
Scrumban Aurora — Gestión ágil de proyectos para LATAM

## Descripción corta (una línea)
Herramienta open source que combina Scrum + Kanban en un tablero liviano, en español y con almacenamiento local.

## Descripción larga (para la página de OpenCollective)

Scrumban Aurora es una herramienta de gestión de proyectos híbrida que combina la planificación estructurada de Scrum con la flexibilidad visual de Kanban. Nacimos para resolver un problema real: los equipos pequeños y freelancers en LATAM no tienen una herramienta ágil que sea liviana, en español y sin la pesadilla de configuración de Jira o los costos de Notion.

### ¿Por qué existe Scrumban?

Porque en Latinoamérica:
- Un equipo de 5 personas no debería pagar $50+/mes por gestionar sus tareas
- No todos tienen internet permanente para usar herramientas cloud
- La mayoría prefiere herramientas en español, no traducciones robóticas
- El setup no debería tomar días — debería tomar segundos

### ¿Qué la hace única?

- **Modo local**: Funciona sin internet, sin cuenta, sin servidor. Tus datos en archivos JSON, tuyos para siempre.
- **Scrumban nativo**: No es un Kanban al que le añadieron sprints después. Fue diseñada desde cero como híbrido.
- **En español de verdad**: Toda la UI, términos ágiles y flujos pensados para LATAM.
- **Zero config**: `npm install && npm start` y ya tienes tu tablero funcionando.
- **Multiplataforma**: Web + Desktop (Electron) + PWA.
- **IA integrada**: Sugerencias inteligentes para escribir mejores historias de usuario.

### Para quién es

- Freelancers que necesitan organizar múltiples proyectos sin pagar suscripciones
- Startups pequeñas que buscan una alternativa liviana a Jira
- Equipos universitarios aprendiendo metodologías ágiles
- Consultorías ágiles que implementan Scrumban con sus clientes

### Nuestro modelo

La herramienta es y será siempre open source (MIT). El modo local es completamente gratuito. Los ingresos por donaciones y suscripciones Pro (cloud sync) nos permiten mantener el proyecto y mejorar la infraestructura para quienes necesitan colaboración en equipo.

### ¿A dónde va tu donación?

- **Infraestructura cloud** ($57/mes): Servidores para el modo cloud
- **Dominio y email** ($10/mes): scrumban.app y comunicación
- **API de IA** (~$50/mes): Sugerencias inteligentes para usuarios
- **Desarrollo**: Tiempo del mantenedor para nuevas features y bugs

Cada donación, por pequeña que sea, nos ayuda a mantener Scrumban vivo y gratuito para toda la comunidad.
